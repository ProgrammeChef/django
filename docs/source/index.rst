Welcome to django-ledger's documentation!
=========================================

A bookkeeping & financial analysis app for the Django Framework.
It incorporates standard accounting practices into a simple & pythonic API with the support of the Django Framework.

Django Ledger takes advantage of Data Analysis libraries like Pandas & Numpy to programmatically generate time-phased \
recurrent and non-recurrent financial transactions.



**THIS PROJECT IS VERY PREMATURE AND UNDER ACTIVE DEVELOPMENT.**
**NOT SUITABLE FOR PRODUCTION ENVIRONMENTS.**

Django Ledger supports:

- Chart of Accounts.
- Financial Statements (Income Statement & Balance Sheets)
- Entities (LLC, Corps, etc.)
- General Ledgers
- Journal Entries
- Accounts

Want to contribute? Don't hesitate to contact me.
